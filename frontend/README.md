# Base React + Vite (JavaScript)

Cực nhẹ để bắt đầu, rồi nâng cấp dần.

## Chạy nhanh
```bash
npm install
npm run dev
# build production:
npm run build
npm run preview
```

## Cấu trúc
```txt
.
├─ index.html
├─ package.json
├─ vite.config.js
└─ src/
   ├─ App.jsx
   ├─ main.jsx
   └─ index.css
```
